> Real-time hands-free computer control using gaze detection and computer vision

- Built for accessibility — enables users to control a computer using only eye movement
- Uses **MediaPipe** for facial landmark detection and **OpenCV** for real-time processing
- React-based frontend interface with Python backend and REST APIs

`Python` `OpenCV` `MediaPipe` `React` `APIs`
