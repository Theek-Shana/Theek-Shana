### 👁️ Eye-Tracking Assistive System
> *Accessibility · Computer Vision · Real-Time*

Built a real-time system enabling **hands-free computer control** through gaze detection — designed to empower users with physical disabilities. Uses facial landmark detection via MediaPipe and real-time frame processing with OpenCV.

### 📚 Online Book Store
> *Full-Stack · Java · Database Design*

Complete bookstore platform with user authentication, book catalog management, and a clean MySQL relational schema. Backend powered by Java Servlets.
